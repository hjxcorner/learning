## 简介

history对象保存着用户上网的历史记录。

借由用户访问过的页面列表，在不知道实际url的情况下实现后退和前进。

| 属性名  | 参数          | 描述                                                         |
| ------- | ------------- | ------------------------------------------------------------ |
| go      | 整数值/字符串 | 向前/向后跳转一个整数值个页面，传入字符串则跳转到历史记录中包含该字符串的第一个位置 |
| forward | 无            | 向前跳转一个页面                                             |
| back    | 无            | 向后跳转一个页面                                             |
